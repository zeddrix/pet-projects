Projects

What is pet-projects?

This monorepo—pet-projects—collects several of my smaller experiments and learning demos from over the years. You can browse the source on GitHub at github.com/zeddrix/pet-projects, or open the live playground at zeddrix.github.io/pet-projects to click through each project in the sidebar.

What you will find

Inside the playground you will find static front-end toys, archived full-stack apps, and README-only references. They are not all production systems; many exist because I wanted to try an idea, finish a tutorial, or preserve code I once shipped elsewhere. Think of this folder as a workshop shelf rather than a product catalog.

Shipped work elsewhere

For more shipped, high-grade products—the kind I have refined with teams and real users—please visit my GitHub profile at github.com/zeddrix and my portfolio at zeddrix.github.io/portfolio. Those sites highlight the work I have been doing more recently, with cleaner architecture and longer maintenance stories.
