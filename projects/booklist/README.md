# Book List

> Browse and manage a personal book collection in the browser.

|                  |                                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------------------ |
| **Developed**    | November 2020                                                                                    |
| **Type**         | Static web demo                                                                                  |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/booklist)              |

Add books with title, author, and ISBN-style fields, then view or remove entries from a table. The app uses ES6 classes for structure and `localStorage` for persistence. An ES5 version of the script is included for comparison.
