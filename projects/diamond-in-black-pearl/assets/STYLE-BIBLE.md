# Style Bible — Diamond in Black Pearl Visual Mode

## Art direction

Storybook adventure — warm painted watercolor feel, soft outlines, gentle texture. Rich but not photorealistic.

## Locked palette

| Role | Hex | Use |
|------|-----|-----|
| Jungle green | `#2d5a3d` | Foliage, paths |
| Lagoon teal | `#3a8f8f` | Water, sky accents |
| Sunlit gold | `#d4a853` | Light, treasure highlights |
| Ship wood | `#6b4423` | Boats, chests, decks |
| Dusk purple | `#4a3f6b` | Shadows, night mood |
| Pink-magenta | `#e056a0` | Panther jewel motif only |

## Composition rules

- 16:9 landscape
- Clear focal subject centered or upper-center
- **Empty lower third** reserved for UI choice buttons
- No text, UI chrome, watermarks, or logos in artwork

## IP safety

- Use **generic** swashbuckling pirates — no actor likenesses
- Use **pink jewel panther statue** — not copyrighted Pink Panther character design
- Ship may be named Black Pearl in game text only — art shows generic pirate galleon
