# GitHub Finder (JSX)

> Search GitHub users with a React interface, client-side routing, and Context API state.

|                  |                                                                                                              |
| ---------------- | ------------------------------------------------------------------------------------------------------------ |
| **Developed**    | November 2020                                                                                                |
| **Type**         | Static web demo (Create React App build)                                                                     |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/github-finder-jsx)                |

A React 17 rewrite of GitHub Finder that uses Context API and hooks instead of Redux. Axios loads user and repo data from GitHub; React Router handles Home, About, and profile routes. The monorepo serves a production CRA build (HashRouter) in the playground iframe; original `src/` is preserved in git history.
