# GitHub Finder

> Search GitHub users and view profiles and public repos with vanilla JavaScript.

|                  |                                                                                                        |
| ---------------- | ------------------------------------------------------------------------------------------------------ |
| **Developed**    | November 2020                                                                                          |
| **Type**         | Static web demo                                                                                        |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/github-finder)              |

Type a GitHub username to fetch profile data and repositories from the public REST API. This was the first version of the project; the React rewrite lives in [GitHub Finder (JSX)](../github-finder-jsx/README.md).

**Status:** Deprecated — prefer [github-finder-jsx](https://zeddrix.github.io/pet-projects/project/github-finder-jsx) for the maintained demo.
