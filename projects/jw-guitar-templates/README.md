# JW Guitar Templates

> A guitar chord and template reference tool with custom typography and printable layouts.

|                  |                                                                                                              |
| ---------------- | ------------------------------------------------------------------------------------------------------------ |
| **Developed**    | March 2020                                                                                                   |
| **Type**         | Static web demo                                                                                              |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/jw-guitar-templates)                |

Built as a personal reference for guitar chord diagrams and template sheets. Custom web fonts and a structured layout make the content readable on screen and suitable for print-style use.
