# Loan Calculator

> Calculate loan payments and amortization schedules in the browser.

|                  |                                                                                                          |
| ---------------- | -------------------------------------------------------------------------------------------------------- |
| **Developed**    | November 2020                                                                                            |
| **Type**         | Static web demo                                                                                          |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/loan-calculator)                |

Enter principal, interest rate, and term to see monthly payment and payoff details. All math runs client-side with vanilla JavaScript — no server or API calls.
