# Robot Friend

> A playful robot figure built entirely with HTML and CSS — no JavaScript required.

|                  |                                                                                                        |
| ---------------- | ------------------------------------------------------------------------------------------------------ |
| **Developed**    | February 2020                                                                                          |
| **Type**         | Static web demo                                                                                        |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/robot-friend)               |

This demo practices CSS positioning and box modeling by assembling a robot character from nested `div` elements. Each body part is styled with borders, backgrounds, and transforms to create a simple interactive-looking figure.
