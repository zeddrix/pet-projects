# Task List

> A simple to-do list that saves tasks in the browser between visits.

|                  |                                                                                                    |
| ---------------- | -------------------------------------------------------------------------------------------------- |
| **Developed**    | November 2020                                                                                      |
| **Type**         | Static web demo                                                                                    |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/tasklist)               |

Add, complete, and remove tasks with immediate UI feedback. Tasks persist via `localStorage`, so the list survives page reloads without a backend.
