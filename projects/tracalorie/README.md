# Tracalorie

> Track daily calorie intake and meals with a simple food log.

|                  |                                                                                                    |
| ---------------- | -------------------------------------------------------------------------------------------------- |
| **Developed**    | November 2020                                                                                    |
| **Type**         | Static web demo                                                                                    |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/tracalorie)             |

Log meals with name and calorie values, edit entries, and see running totals. Data is stored in `localStorage` so the tracker works offline after the first load.
