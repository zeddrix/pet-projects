# Weather Widget

> Display current weather conditions in a compact, location-aware widget.

|                  |                                                                                                          |
| ---------------- | -------------------------------------------------------------------------------------------------------- |
| **Developed**    | November 2020                                                                                            |
| **Type**         | Static web demo                                                                                          |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/weather-widget)                 |

Pick a city and state to load live conditions from the OpenWeatherMap API. Saved locations persist in `localStorage` so returning visitors see their last place by default.
