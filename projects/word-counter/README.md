# Word Counter

> Count words, characters, and reading time for text as you type.

|                  |                                                                                                        |
| ---------------- | ------------------------------------------------------------------------------------------------------ |
| **Developed**    | June 2020                                                                                              |
| **Type**         | Static web demo                                                                                        |
| **Live preview** | [pet-projects playground](https://zeddrix.github.io/pet-projects/project/word-counter)                 |

A lightweight writing utility that updates stats in real time from a textarea. The layout uses BEM-style class names and responsive CSS so the tool stays usable on smaller screens.
